# Day 02 — Array Basics

Practice creating and manipulating arrays with built-in methods.

## Instructions

### Challenge 1: `getFirst(arr)`
Return the first element of the array

### Challenge 2: `getLast(arr)`
Return the last element of the array (don't hardcode the index!)

### Challenge 3: `reverseArray(arr)`
Return a **new** reversed array (don't mutate the original)

### Challenge 4: `sumAll(arr)`
Return the sum of all numbers in the array

### Challenge 5: `includesValue(arr, val)`
Return `true` if the array contains the value, `false` otherwise

### Challenge 6: `removeFirst(arr)`
Return a **new** array without the first element (don't mutate the original)

**Tip:** Remember the difference between methods that mutate and those that return new arrays!
