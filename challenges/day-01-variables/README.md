# Day 01 — Variables & Data Types

Today's challenge covers the fundamentals of JavaScript variables and data types.

## Instructions

Complete each function in the editor according to its description.
Run the tests to check your work, then submit when you're satisfied.

### Challenge 1: `greet(name)`
Return a greeting string like `"Hello, Ana!"`

### Challenge 2: `getType(value)`
Return the type of the value as a string (use `typeof`)

### Challenge 3: `toNumber(str)`
Convert a string to a number and return it

### Challenge 4: `isEven(n)`
Return `true` if the number is even, `false` otherwise

### Challenge 5: `getFullName(first, last)`
Return the full name with a space between first and last

### Challenge 5: `appendToString()`

The function should return a new string which consists of the second string appended to the first string.

Examples:

```js
appendToString('Hello', ' World!'); // "Hello World!"
appendToString('Foo', 'bar'); // "Foobar"
appendToString('bar', 'Foo'); // "barFoo"
appendToString('', 'test'); // "test"
appendToString('other test', ''); // "other test"

**Tip:** Use template literals where possible!
