# Day 01 — Variables & Data Types

Today's challenge covers the fundamentals of JavaScript variables and data types.

## Instructions

Complete each function in the editor according to its description.
Run the tests to check your work, then submit when you're satisfied.

### Challenge 1: `greet(name)`
Return a greeting string like `"Hello, Ana!"`

### Challenge 2: `getType(value)`
Return the type of the value as a string (use `typeof`)

### Challenge 3: `toNumber(str)`
Convert a string to a number and return it

### Challenge 4: `isEven(n)`
Return `true` if the number is even, `false` otherwise

### Challenge 5: `getFullName(first, last)`
Return the full name with a space between first and last

### Challenge 6: `appendToString(first, second)`
The function should return a new string which consists of the second string appended to the first string.

### Challenge 7: `charAt`
Write a function called charAt which accepts a string and an index (number) and returns the character at that index.
The function should return an empty string if the number is greater than the length of the string.
Do not use the built in **charAt** method

### Challenge 8: isAlt
Create a function isAlt that accepts a string as an argument and validates whether the vowels (a, e, i, o, u) and consonants are in alternate order.

### Challenge 9: removeVowels
Write a function called removeVowels which will accept a string and return a new string with all the vowels removed. You should not consider "y" to be a vowel.

### Challenge 10: removeFromString
Write a function called removeFromString, which accepts a string, a starting index (number) and a number of characters to remove.
The function should return a new string with the characters removed.

**Tip:** Use template literals where possible!
